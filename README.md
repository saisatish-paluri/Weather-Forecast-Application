Weather Forecast Application

This is a simple weather application built using React, TypeScript, and Vite. The app allows users to enter a city name and view the current weather information such as temperature, humidity, and weather conditions. It uses a public weather API to fetch real-time data.

Features

* Search weather by city name
* Shows temperature, humidity, and basic weather details
* Fast and lightweight interface
* Built with React + TypeScript + Vite

Technologies Used

* React
* TypeScript
* Vite
* Weather API
* CSS


About

This project was created to learn how to work with React, TypeScript, API integration, and modern frontend tools. It provides a simple way to view live weather information for any city.
